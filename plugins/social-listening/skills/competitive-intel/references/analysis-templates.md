# Competitive Intelligence Analysis Templates

## Template 1: Quick Competitive Pulse

Use this for a fast competitive check (15-minute analysis).

```
## Competitive Pulse — [Date]

### [Competitor Name]

**Recent activity (last 7 days):**
- Posts published: [count by platform]
- Top post: [summary + link + engagement]
- Key topics: [list]

**Sentiment snapshot:**
- Positive mentions: [count]
- Negative mentions: [count]
- Notable: [any standout mentions]

**What they're pushing:** [main message/product/feature]

**Our take:** [one-sentence strategic implication]
```

## Template 4: Product Launch Tracker

Use this when a competitor launches or announces something.

```
## Competitor Launch Alert: [Competitor] — [Product/Feature]

### What They Announced
[Summary of the launch/announcement]

### How They Announced It
- Platform(s): [where]
- Format: [how — blog, tweet thread, video, etc.]
- Messaging: [key claims and positioning]

### Market Reaction
- **Volume:** [number of posts/mentions about it]
- **Sentiment:** [positive/negative/mixed]
- **Key reactions:**
  - "[Positive reaction quote]" — [Author, Platform]
  - "[Negative reaction quote]" — [Author, Platform]
  - "[Neutral/analytical reaction]" — [Author, Platform]

### Competitive Impact
- Does this affect our positioning? [yes/no + how]
- Does this address a gap we were exploiting? [yes/no]
- Does this create a new competitive dynamic? [yes/no + what]

### Recommended Response
- **Sales team:** [what to tell prospects]
- **Marketing team:** [content/messaging response]
- **Product team:** [feature/roadmap implications]
- **Timeline:** [how quickly to respond]
```
